### Project Description
Students must select one of the following systems and implement it using Java classes and inheritance.
Each project must include:
* • At least one base (parent) class
* • At least two child (sub)classes
* • Use of:
* o attributes
* o methods
* o constructors
* o inheritance (extends)
* o object creation
